# Joshua Jumao-as Portfolio

Premium developer portfolio for Joshua Jumao-as, a Computer Engineering student and aspiring full-stack software engineer from Cebu, Philippines.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- ShadCN-style UI primitives
- Lucide React

## Local Development

```bash
npm run dev
```

Open `http://localhost:3000`.

## Production Checks

```bash
npm run lint
npm run build
```

## Content

Main portfolio content lives in `src/data/portfolio.ts`. Update the contact links, project repository URLs, demo URLs, and resume file there before deployment if Joshua uses different accounts.
